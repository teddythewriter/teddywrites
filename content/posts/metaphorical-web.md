I'm stuck.

I'm stuck in a metaphorical web I created myself.

I'm stuck in need of constant validation to keep my motivation up and running.

I'm stuck in this self-made web of self constraints.

Did you know that spiders make these webs out of their Ass? I was probably thinking out of my ass when I created this metaphorical web as well.

It doesn't change the fact that I'm still stuck.

It's not that bad though, The web looks nice when an occasional compliment hangs off of it like a dew drop.

I find it comforting.

I feel like I'm suffering from Stockholm syndrome. I'm in love with this self-made web of constraints. This is my comfort zone, I don't want to leave it.

I'm fine here.

I'm invincible here.
